# tweetpy
develop Ubuntu commands for Twitter with Python


## how-to
### 初回
1. 次のライブラリをインストール
  * requests
  * requests-oauthlib

```
sudo pip3 install requests requests-oauthlib
```

2. config.py
config.pyに次の4つのキーを埋め込む
  * CONSUMER_KEY
  * CONSUMER_SECRET
  * ACCESS_TOKEN
  * ACCESS_TOKEN_SECRET


### 実行
1. ツイート
ツイートをします。

```
> python3 tweet.py
内容を入力してください。
>> ここにツイート
*******************************************
Success.
>
```
